version 3 video+image mixer
{
  "type": "video",
  "video_path": "path/to/video.mp4",
  "duration": 5,
  "video_start": 0,
  "video_end": 10,
  "video_speed": 1.0,
  "resize_method": "crop",
  "background_effect": {
    "type": "zoom",
    "zoom": 1.05,
    "direction": "in"
  }
}

Video-Specific Options:
video_start: Start time in source video (seconds)
video_end: End time in source video (optional)
video_speed: Playback speed (1.0 = normal, 2.0 = 2x, 0.5 = slow motion)
resize_method: How to fit video ("crop", "fit", "stretch")
